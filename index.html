Usage

  1. Create a new project
  cargo new "${NAME}"
  cd "${NAME}"
  1. Merge this repo with --allow-unrelated-histories. This way, we can pull in updates later.
  git remote add template https://github.com/adam-gaia/rust-template
  git fetch template
  git merge --allow-unrelated-histories template/main

Cloning this repo

When cloning this repo, set up to pull CI files from https://github.com/epage/_rust

  git remote add ci https://github.com/epage/_rust.git
  git fetch ci
  git merge --allow-unrelated-histories template/main

Grab the files in .github/workflows/

Notes

The merging unrelated history process was inspired by jonhoo's rust ci repo.
